# Exercise 00 – Sum of Digits

## Objective

Calculate the **sum of digits** of a six-digit integer.

The number is hardcoded directly in the source code via explicit variable initialization.

## Constraints

- Language: Java (latest LTS)
- Only **primitive types** may be used
- Only **standard primitive operators** allowed
- Output via `System.out`
- No user input, no arrays, no methods outside `main`

## Example

```
$> java Program
42
```

*(The number `479598` yields `4+7+9+5+9+8 = 42`)*

## Key Concepts

- Primitive type `int`
- Modulo (`%`) and integer division (`/`) to extract digits
- Accumulation with a loop or sequential operations
